# Notes and Reporting application for an NGO

Starter project for building and deploying a web app with Back4App and Netlify.

## What is set up so far

- GitHub repository connected
- Netlify config file added (`netlify.toml`)
- Environment variables template created (`.env.example`)
- Local Vite setup with a simple `index.html`

## Local development

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
